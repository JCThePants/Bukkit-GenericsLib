Detailed Description

Interface for the global scoreboard tracker.

Tracks the scoreboards that players have viewed so they can be re-shown.

Helps reduce plugin scoreboard conflicts by acting as the central scoreboard manager. Each plugin can simply apply their scoreboards via IScoreboardManager to ensure that when a plugin is done showing a scoreboard to a player, the previous scoreboard is re-shown.
