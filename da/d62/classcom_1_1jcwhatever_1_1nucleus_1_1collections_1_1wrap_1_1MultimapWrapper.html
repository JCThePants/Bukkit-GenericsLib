Detailed Description

An abstract implementation of a synchronized Multimap wrapper.

The wrapper is optionally synchronized via a sync object or ReadWriteLock passed into the constructor using a SyncStrategy.

If the map is synchronized, the sync object must be externally locked while any associated iterator is in use. Otherwise, a java.lang.IllegalStateException will be thrown.

The actual map is provided to the abstract implementation by overriding and returning it from the map method.

In order to make using the wrapper as an extension of a map easier, several protected methods are provided for optional override. See onPut, onPutAll, onRemove, onRemoveAll,@link#onClear</p>
