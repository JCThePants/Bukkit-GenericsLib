Detailed Description

Interface for an object that accepts script event subscribers.

The implementing type represents the scope of the subscription. For example, a subscriber attached to a INpcRegistry will receive updates for all NPC's created by the registry whereas a subscriber attached to an INpc will receive updates only for the NPC it is attached to and only for the lifetime of the NPC.
