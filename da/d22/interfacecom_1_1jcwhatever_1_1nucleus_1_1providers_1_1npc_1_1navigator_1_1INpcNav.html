INpcNavSettings getCurrentSettings ( )

Get the navigator settings for the current pathing target.

When a target is set, the current settings are the settings available from the method getSettings. Changes to the current settings only last until the current navigation ends.

Current settings are used to make adjustments to the settings that are transient. Use the non-current settings from getSettings to retrieve base settings.
