1 NucleusFramework for Spigot 1.8
2 ==================
3 
4 A plugin framework and rapid development library for Bukkit/Spigot. Developed concurrently with the Tales of Arboria server allowing the framework to solve real world development and administrative problems.
5 
6 Currently under development, expect breaking changes.
7 
8 Compatible with Spigot 1.8.8
9 
10 _Coders like versatility. Server operators like stability. Users like usability._
11 
12 ## Goals
13  * Provide a centralized framework for common plugin utilities and services.
14  * Reduce the amount of redundant code in plugins.
15  * Speed up plugin development by abstracting common plugin features.
16  * Create a user command interface that has commonality that users can learn to recognize.
17  * Reduce server operator work by centralizing features that are used across plugins.

23 ## Resources
24  * [NucleusLocalizer](https://github.com/JCThePants/NucleusLocalizer) - A console program used to generate language localization resource files for Nucleus based plugins.
25  * [ResourcePackerMC](https://github.com/JCThePants/ResourcePackerMC) - A console program that aids in generating resource packs. Also generates resource-sounds.yml file used by NucleusFramework to define resource pack sounds.
26  * [Scripting examples](https://github.com/JCThePants/NucleusScriptExamples) - Examples of cross plugin scripts using NucleusFramework.
27  * [NpcTraitPack](https://github.com/JCThePants/NpcTraitPack) - A collection of NPC traits that can be used with a NucleusFramework NPC provider.

29 ## Plugins using NucleusFramework
30  * [PV-Star](https://github.com/JCThePants/PV-Star) - Extensible arena framework.
31  * [TPRegions](https://github.com/JCThePants/TPRegions) - Portal and region teleport.

34  * [RentalRooms](https://github.com/JCThePants/RentalRooms) - Rented regions that allow the tenant to only modify the interior of the house/room.
35  * [RemoteConsole](https://github.com/JCThePants/RemoteConsole) - A remote console for Bukkit/Spigot servers.
