Detailed Description

Used to pass in an com.jcwhatever.nucleus.utils.observer.event.IEventSubscriber from a script using IScriptEventSubscriber.

An IScriptEventSubscriber can be passed in from a script and encapsulated by ScriptEventSubscriber to convert it to an EventSubscriber.
