Detailed Description

An abstract implementation of an ISubscriberAgent.

Provides the basic implementation of tracking and removing subscribers and handling disposal.

boolean registerReference ( ISubscriber  subscriber)

Adds a subscriber.

Equivalent to invoking addSubscriber except that the subscribers ISubscriber#registerReference method is NOT invoked.

Should only be invoked by an ISubscriber implementation that already has a reference to the subscriber due to its ISubscriber#subscribe(ISubscriberAgent) method being invoked. Avoid this method for all other contexts.
