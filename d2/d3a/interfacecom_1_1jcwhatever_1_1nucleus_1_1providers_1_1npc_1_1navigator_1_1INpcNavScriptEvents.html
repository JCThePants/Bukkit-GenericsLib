Detailed Description

Interface for an object that accepts NPC navigator script event subscribers.

The implementing type represents the scope of the subscription. For example, a subscriber attached to a com.jcwhatever.nucleus.providers.npc.INpcRegistry will receive updates for all NPC's created by the registry whereas a subscriber attached to a INpc will receive updates only for the NPC it is attached to and only for the lifetime of the NPC.
