Detailed Description

A type that subscribes to events received from an IEventAgent.

Usage is similar in context to an update subscriber except the event subscriber is more complex.

If you're usage context does not require event priorities and event cancelling, it is recommended to use com.jcwhatever.nucleus.utils.observer.update.IUpdateSubscriber in conjunction with com.jcwhatever.nucleus.utils.observer.update.IUpdateAgent instead.
