Detailed Description

A com.jcwhatever.nucleus.utils.astar.IAStarNodeFactory implementation that pools nodes and their coordinates.

Should not be used except in controlled isolation and when warranted. When used, the Coords3Di in the AStarResult are actually pooled MutableCoords3Di instances which may change values the next time an AStar search is invoked using the same AStarPooledNodeFactory. The results of the previous AStar search must no longer be in use before using the pooled factory again.

In addition, if any of the coordinates from the result need to be stored, they should be copied.
