Detailed Description

NPC Trait provider.

Represents a single type of trait. Used to create instances of the trait type for specific INpc instances.

NpcTrait addToNpc ( INpc  npc)

Create a new instance of the trait for a specific INpc instance and attach it.

Attempts to add a pooled trait from the provider. If one is not available or the provider does not implement trait pooling, a new trait is created by invoking createTrait.

Parameters
npcThe npc instance.
Returns
The newly created instance or the one the NPC already had.
