static ItemStack getItemStack ( Object  object)
static

Convert an object to an ItemStack if possible.

The following are valid arguments that can be converted:

  • ItemStack
  • Material
  • MaterialData
  • The name or alternate name of a material. Valid names are from NamedMaterialData.
  • A item stack string serialized by an IItemStackSerializer.
Parameters
objectThe object to retrieve an ItemStack from.
Returns
An ItemStack or null if failed.
