Detailed Description

A com.google.common.collect.SetMultimap wrapper that stores elements using the player ID as key.

When the player logs out, the entry is automatically removed.

Thread safe.

The maps iterators must be used inside a synchronized block which locks the map instance. Otherwise, a java.lang.IllegalStateException is thrown.
