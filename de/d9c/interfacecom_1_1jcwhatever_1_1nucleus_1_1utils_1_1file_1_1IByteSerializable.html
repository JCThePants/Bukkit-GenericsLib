Detailed Description

Mixin used by an IByteWriter to serialize an object and by IByteReader to subsequently deserialize data from a stream.

The implementer must be able to serialize itself into the stream using the provided instance of IByteWriter and deserialize data into an empty instance of itself using the IByteReader.

For de-serialization, the implementer is required to have an empty constructor. The constructor does not have to be public.
