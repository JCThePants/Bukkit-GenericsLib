Detailed Description

A synchronized extension to Bukkit's org.bukkit.Location class that is partially thread safe.

It's not thread safe when getting Bukkit objects such as org.bukkit.block.Block, org.bukkit.World or org.bukkit.Chunk.

Useful for creating/loading a org.bukkit.Location from an asynchronous thread where it's not safe to retrieve the Bukkit org.bukkit.World object. The SyncLocation can hold the name of the world until it's safe to retrieve the Bukkit org.bukkit.World (safety is determined by the coder using the class).

When using Location#equals on a SyncLocation, the result is always false. however when reversed, a SyncLocation will be able to compare itself to a Location instance.
