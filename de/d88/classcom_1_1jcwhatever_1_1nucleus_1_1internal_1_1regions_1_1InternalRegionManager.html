Detailed Description

Global Region Manager.

Tracks instances of IRegion and provides methods to determine which regions a player is in as well as track players to determine when they enter and leave player watcher regions.

Methods that return IRegion instances are returning com.jcwhatever.nucleus.regions.ReadOnlyRegion instances. This is to prevent inter-plugin conflicts caused by changes to a region that the region owning plugin is unaware of.

Methods that return a specific region type return the actual region instance.
