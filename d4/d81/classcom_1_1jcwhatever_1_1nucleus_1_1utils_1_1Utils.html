Class< T >  clazz

static

Unwrap a potentially wrapped object.

If the object implements IWrapper and the wrapped instance is of the expected type, the encapsulated object is returned.

If the object does not implement IWrapper but the object is of the expected type, the object is returned.

If the encapsulated object and the object are not of the expected type, null is returned.

objectThe potential wrapper.
