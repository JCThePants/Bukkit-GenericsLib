Detailed Description

An implementation of TimedMultimap that utilizes an internal Multimap with hash keys and hash set values.

Thread safe.

The sets iterators must be used inside a synchronized block which locks the set instance. Otherwise, a java.lang.IllegalStateException is thrown.
