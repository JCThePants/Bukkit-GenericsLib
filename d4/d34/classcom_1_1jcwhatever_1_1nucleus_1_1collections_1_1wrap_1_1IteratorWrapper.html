Detailed Description

An abstract implementation of a synchronized Iterator wrapper.

The wrapper is optionally synchronized via a sync object or ReadWriteLock passed into the constructor using a SyncStrategy.

If the iterator is synchronized, the sync object must be externally locked while the iterator is in use. Otherwise, a java.lang.IllegalStateException will be thrown.

The actual iterator is provided to the abstract implementation by overriding and returning it from the iterator method.

In order to make using the wrapper as an extension of a iterator easier, the onRemove method is provided for optional override.
