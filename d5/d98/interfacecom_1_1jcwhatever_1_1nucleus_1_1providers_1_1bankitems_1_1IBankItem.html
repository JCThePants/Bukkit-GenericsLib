Detailed Description

An IBankItem represents all items of the same type, meta, durability, etc.

in a players account. The amount of the item can be "infinite" (or up to the technical limit).

The IBankItem may also represent a sub item of the root IBankItem that is used to represent an ItemStack in an amount up to its maximum stack size.

The IPaginator implementation as well as the Iterable implementation returns IBankItem's that are sub items of their root IBankItem instance. Each sub item can have no more than the maximum stack size of the item type. When a sub item is modified, it also modifies the root item.

Note that the IBankItem is not publicly modifiable. Any iterators or collections produced by the item should honor this. Use java.lang.UnsupportedOperationException where appropriate. Modifications should take place through IBankItemsAccount.
