Detailed Description

An abstract implementation of a Entry wrapper designed to convert between the entries internal value type and the publicly visible value type.

The wrapper is optionally synchronized via a sync object or ReadWriteLock passed into the constructor using a SyncStrategy.

The actual entry is provided to the abstract implementation by overriding and returning it from the entry method.
