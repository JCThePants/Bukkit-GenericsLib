Detailed Description

Abstract implementation of a region.

The region is registered with NucleusFramework's IGlobalRegionManager as soon as it is defined (P1 and P2 coordinates set) via the regions settings or by invoking the setCoords method.

The regions protected methods onPlayerEnter and onPlayerLeave are only invoked by the global region manager if the implementing type invokes setEventListener(boolean) with a 'true' argument.
