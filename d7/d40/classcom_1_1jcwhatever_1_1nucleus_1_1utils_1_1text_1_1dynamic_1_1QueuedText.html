Detailed Description

An IDynamicText implementation that can be used to queue text to be displayed for specified amounts of time in the order that they are queued.

Can also add java.lang.Runnable's to the queue as well as insert pauses between queued text and java.lang.Runnable's.

Member Function Documentation

int getRefreshRate ( )

The preferred (but not guaranteed) speed the text should be refreshed at measured in ticks.

The value returned may change depending on how often the text needs to be refresh at any given time.

A return value of 0 or less indicates the interval is unimportant and/or not required.
