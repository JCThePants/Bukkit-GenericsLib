Detailed Description

A HashMap based implementation of a SubscriberMap which automatically removes subscribers when they are disposed.

Thread safe.

The maps iterators must be used inside a synchronized block which locks the map instance. Otherwise, a java.lang.IllegalStateException is thrown.
