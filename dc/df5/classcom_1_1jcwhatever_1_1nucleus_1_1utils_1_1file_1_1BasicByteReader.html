Bytes from the stream need to have been generated by BasicByteWriter in order to be read.

Get a boolean.

Booleans read sequentially are read as bits from the current byte. When the byte runs out of bits, the next bit is read from the next byte.

Bytes that store boolean values do not share bits with other data types.

ItemStack getItemStack ( ) throws IOException

Get the next group of bytes as an item stack.

Reads the item stack as follows:

  • Boolean (bit or byte depending on the data structure) indicating if the item stack is null. 0 = null. (See getBoolean)
  • Material - Enum (See getEnum)
  • Durability - Integer (See getInteger)
  • Meta count - Integer (See getInteger)
  • Meta collection

Meta is read as follows:

  • Meta Name - UTF-8 String byte array preceded with a byte to indicate length.
  • Meta Data - UTF-16 String (See getString)
