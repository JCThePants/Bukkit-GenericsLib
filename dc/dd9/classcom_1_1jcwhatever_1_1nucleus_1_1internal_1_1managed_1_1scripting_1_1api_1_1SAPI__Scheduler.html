IScheduledTask runTaskLater ( int  delay,
final Runnable  runnable 
)

Run a task after a specified number of ticks have elapsed.

A TaskHandler instance can be used in place of a java.lang.Runnable to add the ability to cancel the task from within the task handler and to run optional code if the task is cancelled.

Parameters
delayThe number of ticks to wait before running the task.
runnableThe java.lang.Runnable to run later.
Returns
A IScheduledTask instance to keep track of the task.

Run a task on a repeating schedule after a specified number of ticks have elapsed and repeating after the specified repeat ticks have elapsed.

A TaskHandler instance can be used in place of a java.lang.Runnable to add the ability to cancel the task from within the task handler and to run optional code if the task is cancelled.

Parameters
initialDelayThe number of ticks to wait before first running the task.
repeatDelayThe number of ticks to wait between each repeat of the task.
runnableThe java.lang.Runnable to run later.
Returns
A IScheduledTask instance to keep track of the task.
