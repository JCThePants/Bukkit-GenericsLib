Detailed Description

A subscriber agent that collections can use for the purpose of automatically removing elements that are disposed.

The CollectionAgent is a wrapper in case an implementation of the collection needs to add its agent to a super class that uses CollectionAgent.

Adds a subscriber.

Equivalent to invoking addSubscriber except that the subscribers ISubscriber#registerReference method is NOT invoked.
