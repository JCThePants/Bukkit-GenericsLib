void write ( boolean  booleanValue) throws IOException

Write a boolean value.

Booleans written sequentially are written as bits into the current byte. When the byte runs out of bits, the next bit is written to the next byte. Because of this, 7 boolean values written sequentially use only 1 byte of space in the stream instead of 7 bytes.

Bytes that store the boolean values do not share their bits with other data types.

void write ( @Nullable UUID  uuid) throws IOException

Write a UUID.

The UUID is written as 16-7 bytes. A boolean is written first to indicate if the value is null. If the value is not null, the next 8 bytes are the most significant bits while the last 8 bytes are the least significant bits.

void write ( @Nullable ItemStack  itemStack) throws IOException

Write an ItemStack.

Writes the item stack as follows:

  • Boolean (bit or byte depending on the data structure) indicating if the item stack is null. 0 = null. (See write(boolean))
  • Material - Enum (See write(Enum))
  • Durability - Short (See write(int))
  • Meta count - Integer (See write(int))
  • Meta collection

Meta is written first with an integer to indicate the number of meta elements followed by the elements, each with the following format:

  • Meta Name - UTF-8 String preceded with a byte to indicate length. (See writeSmallString(String)
  • Meta Data - UTF-16 String (See write(String))
