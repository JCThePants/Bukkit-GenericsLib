Detailed Description

An abstract implementation of an ISubscriber.

Provides the basic implementation of subscribing to and un-subscribing from agents as well as handling disposal.

boolean registerReference ( ISubscriberAgent  agent)

Add an agent.

Equivalent to invoking subscribe except the added agents ISubscriberAgent#addSubscriber method isn't invoked.

Should only be invoked by an ISubscriberAgent implementation that already has a reference to the subscriber due to its ISubscriberAgent#addSubscriber(ISubscriber) method being invoked. Avoid this method for all other contexts.

boolean unregisterReference ( ISubscriberAgent  agent)

Remove an agent.

Functionally the same as unsubscribe except the agents ISubscriberAgent#unregisterReference method isn't invoked. Should only be invoked by an agent that has already removed its reference to the subscriber.

Should only be invoked by an ISubscriberAgent implementation that already removed its reference to the subscriber due to its ISubscriberAgent#removeSubscriber(ISubscriber) method being invoked. Avoid this method for all other contexts.
