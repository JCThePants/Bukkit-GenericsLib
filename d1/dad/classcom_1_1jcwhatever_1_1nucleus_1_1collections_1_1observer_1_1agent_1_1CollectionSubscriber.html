Detailed Description

A ISubscriber that collections of ISubscriberAgent can use for the purpose of automatically removing elements that are disposed.

The CollectionSubscriber is a wrapper in case an implementation of the collection needs to add its subscriber to a super class that uses CollectionSubscriber.

Add an agent.
