Detailed Description

An element distributor that must be polled for the current element.

The current element changes to the next when its specified time span expires. The distributor is circular, meaning once it reaches the end of the collection it will go back to the beginning.

The current element is passively changed. The current method must be invoked at interval for the distributor to be effective.

Not thread safe.
