Detailed Description

A java.util.Set that allows retrieving the items without iteration.

Implements a java.util.HashMap internally with both key and value set to the same object.

Useful for retrieving items from a collection using a substitute object while maintaining the performance of a java.util.HashMap or java.util.HashSet in most cases.
