IChatMessage generate ( IRegisteredCommand  command,
String  template 
)

Generate command usage using the specified root command name and a formatting template that uses the following parameters:

  • {0} - root command name
  • {1} - command path excluding the root command and the command.
  • {2} - the command name
  • {3} - command parameters

Note that a space is added after values inserted for the above parameters except when the parameter value is empty.

Parameters
commandThe command to generate usage text for.
templateThe format template for the generated text.
