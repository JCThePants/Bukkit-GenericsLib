Detailed Description

A type that observes an agent.

When the type is disposed, it must unregister itself from all of its agents as well as release its agent references.

The subscriber is expected to hold a reference to any and all ISubscriberAgent's that register or add themselves to the subscriber, even if the subscriber implementation is not compatible with the agent implementation. The subscriber must hold the reference and treat it as a member in any ISubscriber methods until the agent is unregistered/removed.

When the subscriber is disposed, it removes itself from all its registered agents by invoking the agents ISubscriberAgent#unregisterReference method.

boolean registerReference ( ISubscriberAgent  agent)

Add an agent.

Equivalent to invoking subscribe except the added agents ISubscriberAgent#addSubscriber method isn't invoked.

Should only be invoked by an ISubscriberAgent implementation that already has a reference to the subscriber due to its ISubscriberAgent#addSubscriber(ISubscriber) method being invoked. Avoid this method for all other contexts.

Remove an agent.
