Detailed Description

Used to serialize entities that are considered part of structures, much like blocks.

Cannot create a org.bukkit.entity.Entity instance but instead retains the information necessary to apply the data to the original block location.

Supported Entity Interfaces:

  • org.bukkit.material.Directional
  • org.bukkit.entity.Painting
  • org.bukkit.entity.ItemFrame
  • org.bukkit.entity.ArmorStand
