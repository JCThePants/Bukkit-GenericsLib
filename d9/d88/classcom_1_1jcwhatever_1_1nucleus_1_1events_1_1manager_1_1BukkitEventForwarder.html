Detailed Description

Used to forward Bukkit events.

The forwarder listens to events that are registered via register and forwards the events to one of the appropriate abstract methods.

An event forwarder can be used to forward the received Bukkit events to another event manager instance that is used in a context that might be interested in the event.

If you need to unregister the forwarder, invoke the dispose method.
