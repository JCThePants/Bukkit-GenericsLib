Detailed Description

Abstract implementation of an NPC Trait.

The name of the trait is the same as the parent NpcTraitType

The NpcTrait can optionally implement the java.lang.Runnable interface. If this is the case, the java.lang.Runnable#run method is called every tick while the NPC is spawned so long as canRun returns true.

The trait may also be reused for a different Npc. The trait is only ever used for one Npc at a time, but can be reused for a different NPC once its current NPC is disposed. If the trait cannot be reused, override the isReusable method and return false.

Whenever the trait is assigned to an INpc, the onAttach method is invoked. This method is intended for optional override and passes the new INpc as argument.

Member Function Documentation

boolean canRun ( )

Determine if the Trait should be run.

Used when the trait implements java.lang.Runnable to determine if the trait wants to be run. This is always checked before java.lang.Runnable#run is invoked.

The default implementation always returns true if the trait implements java.lang.Runnable.
