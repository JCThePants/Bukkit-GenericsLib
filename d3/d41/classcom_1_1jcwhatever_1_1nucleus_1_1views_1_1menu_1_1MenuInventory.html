Detailed Description

An org.bukkit.inventory.Inventory wrapper.

Note that CraftBukkit methods that accept org.bukkit.inventory.Inventory instances cast them to CraftInventory. If the instance is to be passed to a Bukkit method, use com.jcwhatever.nucleus.utils.Utils#unwrap or invoke getHandle to get the CraftInventory instance.
