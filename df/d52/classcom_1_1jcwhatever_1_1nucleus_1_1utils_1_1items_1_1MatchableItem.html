Detailed Description

An org.bukkit.inventory.ItemStack wrapper.

Provides built in ItemStackMatcher support in the equals and hashCode methods making the wrapper ideal for use as a hash key that represents the item.

hashCode method returns the hash based on the encapsulated org.bukkit.inventory.ItemStack's org.bukkit.Material type so that different ItemStackMatcher compare operations can be used to find an org.bukkit.inventory.ItemStack by key or in a hash set.
