Detailed Description

A deque of ISubscriber which automatically removes subscribers when they are disposed.

Assumes the subscriber is properly implemented and calls the ISubscriberAgent#removeSubscriber method of all ISubscriberAgent instances that are registered to it when it's disposed.

The collection has its own internal agent which is used to track the subscribers in the collection. If an implementation wishes to add its own agent, it can do so by passing it into the constructor.

Implementations may need to use their own synchronization object, in which case it can be passed in via the constructor using a SyncStrategy.

The deque iterators must be used inside a synchronized block which locks the deque instance. Otherwise, a java.lang.IllegalStateException is thrown.
