Detailed Description

A com.google.common.collect.Multimap of ISubscriberAgent which automatically removes agents when they are disposed.

Assumes the agent is properly implemented and calls the ISubscriber#unsubscribe method of all com.jcwhatever.nucleus.utils.observer.ISubscriber instances that are observing it when it's disposed.

The map has its own internal subscriber which is used to observe the agents in the map. If an implementation wishes to add its own subscriber, it can do so by passing it into the constructor.

Implementations may need to use their own synchronization object, in which case it can be passed in via the constructor using a SyncStrategy.

The maps iterators must be used inside a synchronized block which locks the map instance. Otherwise, a java.lang.IllegalStateException is thrown.
