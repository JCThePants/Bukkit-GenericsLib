Detailed Description

An abstract implementation of a ListIterator wrapper designed to convert between the encapsulated list element type and an externally visible type.

The wrapper is optionally synchronized via a sync object or java.util.concurrent.locks.ReadWriteLock passed into the constructor using a SyncStrategy.

If the iterator is synchronized, the sync object must be externally locked while in use. Otherwise, a java.lang.IllegalStateException will be thrown.

The actual list is provided to the abstract implementation by overriding and returning it from the iterator method.

The convert and unconvert abstract methods are used to convert between the internal list element type and the external type.

When using the unconvert method, a ClassCastException can be thrown to indicate the value cannot be converted. The exception is caught and handled where it is appropriate to do so.
