static boolean enforceNmsVersion ( Plugin  plugin,
String...  compatibleVersions 
)
static

Enforce the compatible NMS versions on the specified plugin.

If the current NMS version is not compatible, the plugin is disabled.

Server operators can override this in the config of plugins that extend NucleusPlugin.

Parameters
pluginThe plugin to check.
compatibleVersionsThe NMS versions the plugin is compatible with.
Returns
True if the plugin is compatible.
