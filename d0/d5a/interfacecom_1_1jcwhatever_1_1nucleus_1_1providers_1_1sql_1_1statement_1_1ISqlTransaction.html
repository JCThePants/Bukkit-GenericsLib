IFutureResult<ISqlResult> append ( ISqlStatement  statement)

Append an Sql statement.

Parameters
statementThe statement to append.
Returns
A future result. The result only contains data created in the statement added to the transaction. Closing any sql results will not effect other statements.
Exceptions
IllegalArgumentExceptionif the statement is not for the same database as the transaction or the ISqlStatement implementation is not supported.
