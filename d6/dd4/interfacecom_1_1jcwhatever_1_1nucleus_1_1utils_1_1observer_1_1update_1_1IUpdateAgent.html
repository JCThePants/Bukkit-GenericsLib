Detailed Description

An update agent which producers can use to send updates to IUpdateSubscriber's that are registered with the agent.

Usage is similar in context to an event agent except the update agent is simpler.

If you're usage context would be better served with event priorities and event cancelling, it is recommended to use com.jcwhatever.nucleus.utils.observer.event.IEventSubscriber in conjunction with com.jcwhatever.nucleus.utils.observer.event.IEventAgent instead.
