Detailed Description

A generic serializable wrapper for a BlockState that is specifically designed to store tile entity data.

Cannot create a org.bukkit.block.BlockState instance but instead retains the information necessary to apply the data to the original block location.

Supported Tile Entities:

  • org.bukkit.inventory.InventoryHolder
  • org.bukkit.block.CommandBlock
  • org.bukkit.block.CreatureSpawner
  • org.bukkit.block.NoteBlock
  • org.bukkit.block.Sign
  • org.bukkit.block.Skull
