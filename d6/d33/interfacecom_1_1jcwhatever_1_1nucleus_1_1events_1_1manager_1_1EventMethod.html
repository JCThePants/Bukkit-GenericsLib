boolean invokeForCancelled ( )

Determine if the handler should be run even if the event is already cancelled.

This is the opposite of Bukkits "ignoreCancelled" parameter. By default, event handlers are not invoked if a previous handler cancels the event. Setting invokeForCancelled() to true allows the event handler to be invoked even if the event has been cancelled.
