Detailed Description

A map of ISubscriber which automatically removes subscribers when they are disposed.

Assumes the subscriber is properly implemented and calls the ISubscriberAgent#removeSubscriber method of all ISubscriberAgent instances that are registered to it when it's disposed.

The map has its own internal agent which is used to track the subscribers in the map. If an implementation wishes to add its own agent, it can do so by passing it into the constructor.

Implementations may need to use their own synchronization object, in which case it can be passed in via the constructor using a SyncStrategy.

The maps iterators must be used inside a synchronized block which locks the map instance. Otherwise, a java.lang.IllegalStateException is thrown.
