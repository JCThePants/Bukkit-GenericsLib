Detailed Description

Uses reflection to load and save fields from an object into an IDataNode.

The fields that can be serialized must be annotated with DataField.

Reduces boilerplate code needed to load and save an objects settings at the cost of performance.

Only the following field types can be serialized: (Primitive wrappers are not supported)

  • boolean
  • byte
  • short
  • int
  • long
  • float
  • double
  • String
  • instance of Enum
  • Location
  • ItemStack
  • ItemStack[]
  • instance of IDataNodeSerializable
